```

Line: 75 (for work) and 76 (for education)

Find: from-blue-500 to-cyan-500
Replace with: Any Tailwind gradient (e.g., from-red-500 to-pink-500)
```

### Make Animations Faster/Slower
```
File: src/components/ExperienceSection.tsx
Look for: transition={{ delay: index * 0.1 }} or duration: 2

To make FASTER: Decrease the number (0.05 instead of 0.1)
To make SLOWER: Increase the number (0.2 instead of 0.1)
```
